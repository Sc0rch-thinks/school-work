CSS

Linking

There are three ways to insert style scripts, using a external css file, using a internal css file or using inline CSS.

To use the external css file method, the link tag is used. To use the link tag, the link tag goes like (link rel='stylesheet' href='file name'). the benefits of this is that it only requires changes in one file to change the entire site.
